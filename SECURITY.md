# Security Policy

## Supported Versions

Currently only the latest version of DiagView is supported for security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Security issues and bugs should be reported privately by opening a [Security Advisory draft](https://github.com/khadirullah/diagview/security/advisories) on GitHub, or by emailing the project authors if that is not possible.

Please provide:
* A description of the vulnerability.
* Steps to reproduce it.
* A description of the impact.
* Any possible mitigations you might have found.

You should receive a response within 48 hours. If the vulnerability is confirmed, a patch will be prepared and released as soon as possible.
